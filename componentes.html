Em baixo podemos ver os componentes do computador e a sua respetiva caracterização..

Placa Gráfica

A placa gráfica é um componente que serve para processar toda a memória gráfica utilizada pelo computador.

Placa de Rede

A placa de rede é utilizada para que seja possível o sistema ligar-se à rede através de Wi-Fi.

Motherboard

A Motherboard é o componente que interliga todos os outros componentes.

Processador

O Processador é o componente que processa todas as tarefas e processos do computador.

RAM

A RAM (Random Acess Memory) é um tipo de memória que armazena memória temporária de processos em execução.

Fonte de Alimentação

A Fonte de alimentação como o nome indica é o componente que alimenta todos os outros componentes través de corrente elétrica.

Disco Rígido

O Disco Rígido é o componente que aramazena todos os ficheiros e programas "permanentes do computador".

Caixa

A caixa é o componente em que todos os componentes são montados.
