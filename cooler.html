Como Instalar o cooler.

Nas imagens em baixo é explicado detalhadamente como se monta o cooler/dissipador.

1º Passo (Intel)

No caso da Intel, basta alinhar os quatro pinos de mola com os buracos de retenção na motherboard. Pressione o cooler para que cada pino encaixe no buraco, de forma cruzada, até ouvir um clique.

1 º Passo (AMD)

Para instalar um dissipador para processadores AMD, basta alinhar a barra de montagem do dissipador com as duas abas no sistema de retenção da motherboard.
